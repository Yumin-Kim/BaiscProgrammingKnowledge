추상자료형(Abstract Data Type). 기능의 규헌 부분은 나타내지 않고 순수한 기능이 무엇인지 나열한것을 추상 자료형이라고 한다 비유 : 사용 설명서와 같이 기능관 사용 방법을 정의한 것이다 >> 예를 들면 내부적으로 어떤 기능을 실행 했을때 와트나 내부 회로에서 어떻게 작용하는지느 전혀 나오지 않는다 추상 자료형은 구현자와 사용자를 분리해준다. 라이브러리를 가져다 쓰거나 내장 함수를 사용하는것도 추상 자료형이 정의 하기 때문 추상 자료형은 내부 은닉으로 이루어져 있다 -정리- 사용자는 내부 구현이 어떻게 구현이 어떻게 되어 있는지 알필요가 없다.추상 자료형을 보고 목적에 맞게 가져다 사용히면 된다 구현자 입장에서는 구현을 위한 기능과 데이터 집합을 나열하는 것이다. 알고리즘 의 선능 분석 기법 실행 기간을 측정 하는 방법 알고리즘의 복잡도 -시간 복잡도 분석 -공간 복잡도 분석
