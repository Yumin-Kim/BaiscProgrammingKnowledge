Babel 공부 하고 가기

babel은 ES6,ES7 코드를 ECMAScript5코드로 transpliing 하기 위한 도구이다

Babel은 다양한 모듈을 담는 일종의 상자 역할을 하며 코드를 컴파일 하기 위해 작은 모듈(presets,plugins등등 )을 사용한다

  • babel-cli

    • command line을 통해 코드를 transplie(사용할 수 있는 문법으로 변환 )

  • babel-register

    • 각각의 모듈을 결합할때 사용하는 hook(후크) 모듈이다

    production을 위한 모듈이 아니다

  • Configuring Babel

    • 위에 있는 모듈들은 현 디렉토리에서 다른 디렉토리로 이동 시켜 주는 역힐을 합니다 >> 실질적 정보를 전달해줘야 한다 >>이정보는 .babelrc파일을 통해서 전달할수 있다!! 

        
        {
            "presets":[],
            "plugins":[],
        }
        
    구성 되어 있다
  • babel-polyfill

    • babel-polyfill은 ES6 횐경을 제공합니다

    polyfill이란 code 조각으로 런타임에 존재하지 않는 native API의 복사본을 말합니다.

    polyfill이 필요한 곳에 import 'babel-polyfill'; 사용하거나

    webpack을 사용한다면 entry:["babel-polyfill"...] 사용

  • babel-plugins

    • 아직 공식 스펙에서 지원하지 않는 기능들을 transform-plugin을 추가하여 사용할 수 있다.

  • babel-node

    • node.js에서 최신 문법을 사용 할 수 있게 한다 >> nodemon과 같이 사용시 nodemon --exec babel-node 파일명 이런식으로 사용힌다 --exec는 js 파일이 아닌것도 실행 시킨다

    그냥 사용하면 메모리적으로 빌드 하는 시간이 소모 됨으로 따로 빌드 한 후에 script를 사용하여 시작 시킨다
