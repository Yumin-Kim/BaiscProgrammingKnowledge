            Pair p;

            p[1] = 2;

사용후


        
            typedef struct {int x,y;} Pair; // 위에 코드와 다르게 직관성 있게 보여진다

C/C++ 포로그래밍 언어에서 구조화 된 데이터를 처리 할때 struct를 사용하는데 이를 구조체라고 한다 말의 의미가 연결되어 한 덩어리로 처리하는 방식을 말하며 record / object처럼 비슷하게 사용된다 (여러 가지의 변수를 하나로 묶어서 하나의 큰 이름(변수)안에 묶어서 좀 더 가독성 있게 표현 할 수 있다)

구조체는 메모리 할당을 순차적으로 한다.

구조체 가르키기(포인터 포함) 

        
            struct ProductInfo pdi12 = { 123333, 50000 , "감귤" };
            struct ProductInfo *ptr = &pdi12; 
            printf("*번호 :  %d\n",(*ptr).num); //.num으로 .struct에서 저희 해준 변수를 멤버라고 부름
            printf("*번호 :  %d\n",ptr->num);
  • 구조체 재선언 안됨(상수와 같은 성질을 가지고있음)
  • (*ptr) 묶어 주어야지 포인터 전체를 가져와서 .멤버를 입력 할 수 있다.
  • (*ptr).num을 축약하여 ptr->num 이라고 사용될 수 있다.
